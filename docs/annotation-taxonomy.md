# Greenwashing Annotation Taxonomy

Based on TerraChoice "Seven Sins of Greenwashing" (2007/2010)

## Labels

| Label | Code | Definition |
|-------|------|------------|
| Confirmed Greenwashing | `confirmed` | Regulatory enforcement, legal settlement, or retracted claim |
| Suspected Greenwashing | `suspected` | Exhibits >=2 sins but no formal enforcement |
| Clean | `clean` | Transparent, verified, third-party audited claims |

## Greenwashing Sin Categories (TerraChoice)

| Sin | Code | Description |
|-----|------|-------------|
| Hidden Trade-Off | `sin_hidden_tradeoff` | Claim based on narrow attribute ignoring broader impacts |
| No Proof | `sin_no_proof` | Claim without accessible evidence or certification |
| Vagueness | `sin_vagueness` | Poorly defined terms ("eco-friendly", "all-natural") |
| False Labels | `sin_false_labels` | Fake certifications or misleading endorsement imagery |
| Irrelevance | `sin_irrelevance` | True but unhelpful claim (e.g., "CFC-free" after ban) |
| Lesser of Two Evils | `sin_lesser_evil` | Green claim within inherently harmful category |
| Fibbing | `sin_fibbing` | Outright false environmental claim |

## Additional Categories (Academic Extension)

| Category | Code | Description |
|----------|------|-------------|
| Selective Disclosure | `ext_selective` | Only reporting positive metrics, omitting negative |
| Misleading Metrics | `ext_misleading_metrics` | Using incomparable baselines or cherry-picked years |
| Aspirational Without Plan | `ext_aspirational` | Net-zero pledge with no roadmap or interim targets |
